# ease-light-dark-toggle 
A sleek animated light/dark mode toggle component for Ease Motion CSS.

## ✨ Features

- 🌞 Animated Sun ↔ 🌙 Moon toggle
- 🎨 Smooth sliding thumb animation
- 🌗 Light and Dark mode support
- 💾 Remembers the selected theme using `localStorage`
- 📱 Fully responsive design
- ⚡ Built with pure HTML, CSS, and JavaScript
- ♿ Accessible and easy to integrate
- 🎯 Clean, modern UI with smooth transitions

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/AmishaOnMain/Ease-Light-Dark-mode.git
```

### 2. Navigate to the project

```bash
cd Ease-Light-Dark-mode
```

### 3. Open the project

Open `index.html` in your preferred web browser.

---

## 💻 Usage

Link the stylesheet in your HTML file:

```html
<link rel="stylesheet" href="style.css">
```

The component automatically:

- Switches between Light and Dark Mode
- Updates the UI with smooth animations
- Saves the selected theme using `localStorage`
